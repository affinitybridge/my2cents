app.js

Project Codename is a campaign widget for calling elected representatives. Source is available on GitHub.

 
             		           
                               
                            
                 ¶               
               
Project Codename is built on Express. It uses the
Twilio API to make phone calls and the
Open North Represent API to receive
representative information. Campaigns are currently persisted in the
capaigns.js file. Jade is used for templating and
Stylus helps with CSS.

  var output = [];
  for (i in reps.objects) {
    var rep = reps.objects[i];
